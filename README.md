# 🌟 LUMORA

LUMORA is a modern web-based perfume shopping application built with **HTML, CSS, and JavaScript**.  
It provides users with an intuitive interface to browse, view, and purchase products seamlessly.

---

## 🚀 Features

- 🏠 **Home Page:** Displays featured and trending products.
- 👤 **User Authentication:** Login and sign-up pages for user access.
- 🛍️ **Product Listings:** View all available products with images, prices, and descriptions.
- 💳 **Shopping Cart:** Add, remove, or update items in your cart.
- 🧾 **Payment Page:** Simple checkout flow for completing purchases.
- ⚙️ **Profile Management:** Edit and update user profiles.
- 💡 **Responsive Design:** Optimized for desktop and mobile devices.

## 🧠 Technologies Used

- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsive design  
- **JavaScript (ES6)** – Interactivity and dynamic content
